This is the codebase for our ECE 482 Senior Design Project
Authors: Matteo De Angelis, Bruce Lozano, Bayden Willms

Rasperry Pi 3 with RFID RC522 chip to automatically take attendance for university courses
    - Webapp for professors to log in and view attendances for their courses
    - Database implementation in MySQL
    - Website hosting using AWS EC2 Instance
    - Webapp backend using a Flask application and Python 3
